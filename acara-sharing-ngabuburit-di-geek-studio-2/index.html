Acara Sharing Ngabuburit di Geek Studio

12 September 2015

Having fun with Java & Git

nah ini dia acara yang watu bulan Ramadhan kemarin kita ngadain acara ngabuburit sambil sharing teknologi di bidang software development.

Kali ini geek studio mau sharing mengenai teknologi java dan versioning control git. Kenapa git? karena sekarang di kampus dan sekolah-sekolah SMK masih saja banyak yang tidak tau mengenai apa sih versioning control system, padahal notabene jurusan mereka adalah Teknik Informatika, Ilmu Komputer, Informatika, atau RPL (Rekayasa Perangkat Lunak).

Nah kali ini kang Eko kurniawan Khannedy Chief Technology Officer (CTO) PT. Starlabs Global Teknologi, sharing mengenai teknologi java kekinian, dan mengambil scala sebagai teknologi yang ingin di perkenalkan.

Setelahnya diselingi dengan pengenalan VCS (Versioning Control System) git oleh Kang Ahmad Arif yang merupakan Instruktur Praktikum dan Gambit Studio.
Gambar : Hendri sedang sharing mengenai Geek Studio
Gambar : Kang Eko lagi sharing mengenai scala

Gambar : temen-temen yang hadir ikut acara ngabuburit. Ini campur anak-anak PKL Animasi cca juga.
setelah itu acara mau mendekati maghrib baru tinggal anak-anak informatika hehee
Gambar : nah yg paling belakang itu, sekumpulan cewek-cewek yang di belakang merupakan siswi dari SMK Al-Itihad Cianjur, yang sekarang sedang PKL juga di Geek Studio.

Nah rame kan di geek studio?? kita belajar dan sharing bareng disini, mengenai software development, startup company, teknologi dan lain-lain soal IT. :D
