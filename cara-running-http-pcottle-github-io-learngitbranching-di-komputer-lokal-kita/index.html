Cara Running http://pcottle.github.io/learnGitBranching/ di komputer lokal kita

14 September 2015

Ok, ini cara sederhana untuk menjalankan learnGitBranching yang merupakan git repository visualizer, untuk memudahkan kita belajar branching di vcs git.

  1. Pertama tentunya kita harus sudah menginstall git di sistem operasi yang kita gunakan. Untuk ubuntu bisa menggunakan : $sudo apt-get install git
  2. Kita harus clone project learnGitBranching di https://github.com/pcottle/learnGitBranching
    • dengan cara : $git clone git@github.com:pcottle/learnGitBranching.git
  3. Karena learnGitBranching menggunakan nodejs makan kita harus menginstall Install nodejs dan npm nya dan grunt :
    • install nodejs di mac : $brew install nodejs
    • sedangkan di ubuntu : $sudo apt-get install nodejs
    • lalu install grunt dengan menggunakan npm : $npm install -g grunt-cli
  4. Setelah itu masuk ke project learnGitBranching yang sebelumnya sudah di clone, lalu ketik :
    • $npm install
    • setelah sukses lalu ketik : $grunt
    • untuk mac mungkin harus menginstall : $brew install watchman
  5. Setelah semua selesai tinggal running project dengan cara : $npm start
    Jika bermasalah coba menggunakan sudo : $sudo npm start

Ketika grunt berjalan...

Setelah npm start berjalan, berarti server sudah up di localhost:8081

tampilan learnGitBranch berjalan di localhost:8081
